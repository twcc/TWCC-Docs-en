# <img class="ccsimgicon" src="https://cos.twcc.ai/SYS-MANUAL/uploads/upload_60c298573392b441958d75106686c11d.png"> PyTorch

TWCC provides 10 ready-to-use developing environments with NGC optmized PyTorch.PyTorch is a tensor computing framework with strong GPU acceleration, and uses Python as the front-end language. Functions can be easily extended by using common Python libraries (such as NumPy, SciPy and Cython). At the functional neural network layer, an automatic differentiation library based on a tape-based system is used to provide a high degree of flexibility and speed for the deep learning framework,  and provide acceleration functions(like NumPy). PyTorch also includes standard-defined neural network, engine performance optimization using machine learning, data loading functions, and supports multiple GPUs and multi-node. The computing can be performed immediately, without waiting in line in a static flow chart, which improves the usability and an excellent debugging experience.
