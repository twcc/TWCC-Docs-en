## Cloud File Service (CFS)

Category of storage permissions:

- <i class="fa fa-eye" aria-hidden="true"></i> View: Browse, download and search files only.
- <i class="fa fa-pencil-square-o" aria-hidden="true"></i> Edit: In addition to View permissions, able to delete, upload, folder create and file transfer between hot and cold strorages.
- <i class="fa fa-share-square-o" aria-hidden="true"></i> Share: Setting folder participator and permissions.


Description betwen storage level and permissions:

| Dataset | Parent folder | Child folder |
| -------- | -------- | -------- |
| File upload to dataset will be able to view and edit by **all project member**.| (1) Permissions of View, Edit and Share only able to **Project Creator** and **Folder Creator**.<br> (2) Participator not eligible to Share permissions.<br>    | Inherit parent folder's permissions setting     |
